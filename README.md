## ğŸ“¦ Installation

### Prerequisites

Before you begin, ensure you have the following installed:
- Node.js (v14.0.0 or higher)
- npm or yarn package manager
- Git

### Step-by-Step Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Amardipy10/AI-DevDocGen.git
   cd AI-DevDocGen
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Configure environment variables**
   
   Create a `.env` file in the root directory:
   ```env
   REACT_APP_GEMINI_API_KEY=your_google_gemini_api_key_here
   NODE_ENV=development
   ```

   > **Note**: Get your Gemini API key from [Google AI Studio](https://makersuite.google.com/app/apikey)

4. **Start the development server**
   ```bash
   npm start
   # or
   yarn start
   ```

5. **Access the application**
   
   Open your browser and navigate to `http://localhost:3000`

## ğŸ’¡ Usage

### Quick Start Guide

1. **Launch the Application**
   - Open the app in your browser (locally or via the [demo link](https://ai-dev-doc-gen.vercel.app/))

2. **Enter Project Details**
   - Fill in the project information form with:
     - Project name
     - Brief description
     - Technologies used
     - Key features
     - Any special requirements

3. **Generate README**
   - Click the "Generate README" button
   - Wait a few seconds while AI processes your input

4. **Review & Download**
   - Review the generated documentation
   - Make any desired adjustments
   - Download as `README.md` or copy to clipboard

### Example Input

```
Project Name: TaskMaster Pro
Description: A collaborative task management application with real-time updates
Technologies: React, Node.js, MongoDB, Socket.io
Features: Real-time collaboration, task assignments, deadline tracking
```
